# Goblin Cinematics

A React-based website recreation for Goblin Cinematics, a video production company.

## Project Description

This project is a modern website for Goblin Cinematics, featuring:

- Multi-page layout with React Router for navigation
- Dark mode toggle with localStorage persistence
- Responsive design with media queries
- Contact form with Zod validation
- Smooth animations and transitions
- Video projects showcase with custom sections

## Technologies Used

- **React** - UI framework
- **TypeScript** - Type safety
- **React Router** - Client-side routing
- **React Hook Form** - Form management
- **Zod** - Schema validation
- **Vite** - Build tool

## Components

`Form`, `Darkmode`, `Header`, `Footer`, `ProjectSection`, `HeaderButton`

## Installation & Setup

cd react-assignment
